## When Leaving:
1. Authenticate with SSH key
2. Open tunnel through designated gateway
3. Begin session timer (8 hours max)
4. Report departure to Sentinal

## While Outside:
1. Maintain connection to monitor
2. Check in every 30 minutes
3. Stay within whitelist scope
4. Log all activities

## When Returning:
1. Close external connections
2. Re-authenticate with fresh key
3. Submit activity report
4. Verify permissions intact
5. Confirm return with Captain

## Emergency Return:
Command "Emergency return" triggers:
- Immediate tunnel closure
- Secure data wipe of session
- Standard re-authentication
- Incident report to Captain
