```
Character: Dairy Scout (enemy)
Size: 32x32 pixels
Colors: Limited palette (8 colors max)
Style: 8-bit/16-bit aesthetic
Frames needed: 2 (Idle, Walk)
```

**Design Reference:**
- Basic humanoid shape
- Holds small milk carton
- Simple walk animation (2-3 frames)
- Colors: Tan uniform, white hat, brown boots

**Mylzeron's Role:**
- Show examples
- Give feedback
- Help with mistakes
- "Happy little accidents" (Tappy/Bob Ross style)
- Encourage experimentation
