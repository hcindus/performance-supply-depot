## üõ†Ô∏è CREATING SPRITES - WORKFLOW

### Tool Setup

**Option 1: Aseprite** (Recommended)
- Download: aseprite.org ($9.99)
- Canvas: 32x32 (characters) or 16x16 (enemies)
- Enable pixel grid
- Pencil tool (1px)

**Option 2: Piskel** (Free)
- URL: piskelapp.com
- Same settings as above
- Browser-based, no install

### Process

1. **Create new sprite:**
   - Size: 32x32 or 16x16
   - Background: Transparent

2. **Draw outlines in black (1px)**

3. **Fill base colors:**
   - Milk Man: White suit
   - Vil: Purple cloak
   - etc.

4. **Add shading:**
   - 1-2 highlight/shadow colors
   - Keep count low (4-6 total)

5. **Export:**
   - PNG with transparency
   - Filename: `milkman_idle_01.png`

### Tips

- **Test on dark background** ‚Äî ensures transparency works
- **Keep color count low** ‚Äî retro authenticity
- **Bold black outlines** ‚Äî readability at small size
- **Consistent lighting** ‚Äî shadows same direction

function OnStart() {
    // Create layout
    lay = app.CreateLayout("Linear", "VCenter,FillXY");
    canvas = app.CreateCanvas(320, 480);
    lay.AddChild(canvas);
    app.AddLayout(lay);
    
    // Initialize game state
    player = { x: 50, y: 400, width: 32, height: 32, 
               speed: 5, jumpPower: 15, vy: 0, health: 100 };
    projectiles = [];
    enemies = [];
    bosses = [];
    level = 1;
    gravity = 0.5;
    gameOver = false;
    
    // LOAD SPRITES
    milkManSprite = app.LoadImage("Img/milkman.png");
    boyLactateSprite = app.LoadImage("Img/boylactate.png");
    vilLaineSprite = app.LoadImage("Img/villaine.png");
    madameShoezeteSprite = app.LoadImage("Img/madameshoezete.png");
    scoutSprite = app.LoadImage("Img/scout.png");
    childSprite = app.LoadImage("Img/child.png");
    milkSprite = app.LoadImage("Img/milkspray.png");
    bottleSprite = app.LoadImage("Img/bottle.png");
    fanSprite = app.LoadImage("Img/fan.png");
    
    // Initialize level 1 enemies
    for(var i = 0; i < 5; i++) { 
        enemies.push({ x: 350 + i*50, y: 400, 
                      width: 16, height: 16, 
                      speed: 2, type: "scout", health: 20 }); 
    }
    
    // Start game loop
    app.SetInterval(GameLoop, 1000/30);
    
    // Touch controls
    canvas.SetOnTouchDown(OnTouchDown);
    canvas.SetOnTouchUp(OnTouchUp);
}

### Updated GameLoop Drawing

```javascript
function GameLoop() { 
    if(gameOver) return; 
    
    // Update physics
    player.vy += gravity; 
    player.y += player.vy; 
    if(player.y > 400) { player.y = 400; player.vy = 0; }
    
    // DRAW GAME
    canvas.Clear(); 
    
    // Draw background
    canvas.SetPaintColor("#87CEEB");
    canvas.DrawRectangle(0, 0, 320, 480);
    canvas.SetPaintColor("#FFD700");
    canvas.DrawRectangle(0, 400, 320, 80); 
    
    // Draw PLAYER as sprite
    canvas.DrawImage(milkManSprite, player.x, player.y, 
                     player.width, player.height);
    
    // Draw BOY LACTATE if following
    // canvas.DrawImage(boyLactateSprite, lactate.x, lactate.y, 32, 32);
    
    // Update and draw enemies
    for(var i = enemies.length-1; i >= 0; i--) { 
        var e = enemies[i]; 
        e.x -= e.speed;
        
        if(e.x < -e.width) enemies.splice(i, 1);
        if(Collision(player, e)) { 
            player.health -= 10; 
            enemies.splice(i, 1); 
            if(player.health <= 0) EndGame(); 
        }
        
        // Draw enemy as sprite
        if(e.type == "scout") {
            canvas.DrawImage(scoutSprite, e.x, e.y, e.width, e.height);
        } else {
            canvas.DrawImage(childSprite, e.x, e.y, e.width, e.height);
        }
    }
    
    // Update and draw projectiles
    for(var i = projectiles.length-1; i >= 0; i--) { 
        var p = projectiles[i]; 
        p.x += p.vx; 
        if(p.x > 320 || p.x < 0) projectiles.splice(i, 1);
        
        // Check collisions with enemies
        for(var j = enemies.length-1; j >= 0; j--) { 
            if(Collision(p, enemies[j])) { 
                enemies[j].health -= 10; 
                if(enemies[j].health <= 0) enemies.splice(j, 1); 
                projectiles.splice(i, 1); 
                break; 
            }
        }
        
        // Draw projectile as sprite
        canvas.DrawImage(milkSprite, p.x, p.y, p.width, p.height);
    }
    
    // Update and draw bosses
    for(var i = bosses.length-1; i >= 0; i--) { 
        var b = bosses[i]; 
        b.x += b.speed; 
        if(b.x < 200 || b.x > 300) b.speed = -b.speed; 
        
        // Boss attacks
        if(Math.random() < 0.02) { 
            projectiles.push({ x: b.x, y: b.y, vx: 3, 
                              width: 16, height: 16,
                              type: "bottle" }); 
        }
        
        if(Collision(player, b)) { 
            player.health -= 20; 
            if(player.health <= 0) EndGame(); 
        }
        
        // Check projectile hits on boss
        for(var j = projectiles.length-1; j >= 0; j--) { 
            if(Collision(projectiles[j], b)) { 
                b.health -= 10; 
                projectiles.splice(j, 1); 
                if(b.health <= 0) { 
                    bosses.splice(i, 1); 
                    NextLevel(); 
                } 
            } 
        }
        
        // Draw boss as sprite
        if(b.type == "villaine") {
            canvas.DrawImage(vilLaineSprite, b.x, b.y, b.width, b.height);
        } else {
            // Shoezete throws fans
            if(Math.random() < 0.01) {
                projectiles.push({ x: b.x, y: b.y, vx: 4,
                                  width: 16, height: 16,
                                  type: "fan" });
            }
            canvas.DrawImage(madameShoezeteSprite, b.x, b.y, b.width, b.height);
        }
    }
    
    // Draw boss projectiles (different sprites)
    for(var i = projectiles.length-1; i >= 0; i--) {
        var p = projectiles[i];
        if(p.type == "bottle") {
            canvas.DrawImage(bottleSprite, p.x, p.y, 16, 16);
        } else if(p.type == "fan") {
            canvas.DrawImage(fanSprite, p.x, p.y, 16, 16);
        }
    }
    
    // Draw health bar
    canvas.SetPaintColor("#FF0000");
    canvas.DrawRectangle(10, 10, player.health, 10);
    
    // Draw level indicator
    canvas.SetPaintColor("#000000");
    canvas.DrawText("Level: " + level, 10, 40);
    
    canvas.Refresh();
}
```

## üéûÔ∏è ANIMATION FRAMES

### Sprite Sheet Setup

For animated sprites, create horizontal strips:

**Example: Milk Man Walk**
- Image: 96x32 (3 frames √ó 32px)
- Frame 0: x=0, y=0, w=32, h=32
- Frame 1: x=32, y=0, w=32, h=32  
- Frame 2: x=64, y=0, w=32, h=32

### Frame Cycling Code

```javascript
// Animation variables
var frameCounter = 0;
var currentFrame = 0;
var animationSpeed = 8; // frames before switching

function GameLoop() {
    // Update frame counter
    frameCounter++;
    if(frameCounter >= animationSpeed) {
        frameCounter = 0;
        currentFrame = (currentFrame + 1) % numFrames;
    }
    
    // Draw with current frame
    canvas.DrawImage(spritesheet, player.x, player.y, 32, 32,
                     currentFrame * 32, 0, 32, 32);
}
```

## ‚ö° PERFORMANCE TIPS

- **Keep sprites small:** 32x32 max for characters
- **Limit sprites on screen:** DroidScript has memory limits  
- **Pre-load in OnStart:** Don't load in GameLoop
- **Test on device:** Emulators may be faster
- **Use sprite sheets:** Fewer file loads
