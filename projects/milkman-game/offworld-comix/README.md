## ğŸ¨ Off World Comix Style

### Visual Identity

**Logo:** Starburst behind milk bottle, "OWC" in retro-futuristic font  
**Colors:** Cream white, dairy blue, heroic gold  
**Font:** Bold sans-serif, slightly rounded (friendly but strong)

### Comic Style

- **Panel Layout:** Dynamic, varied layouts (full page spreads for hero moments)
- **Line Work:** Clean, confident lines (inspired by Jack Kirby meets 50s advertising)
- **Coloring:** Flat colors with subtle gradients (digital-friendly but print-ready)
- **Sound Effects:** BIG, stylized onomatopoeia ("CRASH!", "MOOOO!", "PASTEURIZE!")
- **Caption Boxes:** Classic rectangle with notched corners

### Cartoon Style

- **Animation:** 2D traditional/limited animation (budget-friendly, retro charm)
- **Frame Rate:** 24fps for action, 12fps for dialogue (classic TV animation)
- **Design:** Simplified from comic designs for animation efficiency
- **Backgrounds:** Painted, slightly stylized (50s suburban + futuristic dairy tech)
- **Music:** Chiptune + big band (Reggie/Ronstrapp provides OST)

### Tone: Self-Aware Earnestness

**The Rule:** Take the world seriously, wink at the audience.

**Comedy DNA:**
- Character reactions are genuine
- Situations are absurd
- Never punch down
