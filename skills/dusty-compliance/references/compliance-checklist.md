# Dusty Compliance Checklist

Pre-launch compliance verification.

## Phase 1: Internal Use Only (Current)

- [ ] Employee crypto payment agreements drafted
- [ ] 10% fee structure documented internally
- [ ] VPS terms of service reviewed (no crypto policy violations)
- [ ] Employee tax withholding process defined
- [ ] Incident response plan documented

## Phase 2: Limited Beta (Pre-Public)

- [ ] Terms of Service drafted (lawyer reviewed)
- [ ] Privacy Policy drafted (GDPR/CCPA compliant)
- [ ] Geographic restrictions implemented (block NY, sanctioned countries)
- [ ] Risk disclosure added to signup
- [ ] API rate limiting configured
- [ ] Logging configured (6 month retention minimum)
- [ ] Exchange API ToS compliance verified

## Phase 3: Public Launch

- [ ] FinCEN MSB registration (if custodial)
- [ ] State-by-state money transmitter licenses (if custodial)
- [ ] KYC provider integrated and tested
- [ ] AML compliance program documented
- [ ] Transaction monitoring configured
- [ ] SAR filing process defined
- [ ] User agreement acceptance logged
- [ ] Data retention policy published
- [ ] Breach notification procedures documented
- [ ] Insurance review (cyber liability, errors & omissions)

## Ongoing

- [ ] Monthly regulatory news review
- [ ] Quarterly compliance audit
- [ ] Annual legal review
- [ ] Employee training on crypto compliance
- [ ] API key rotation schedule maintained
