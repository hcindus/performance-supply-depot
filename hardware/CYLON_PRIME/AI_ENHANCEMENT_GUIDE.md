## KAEDIM WORKFLOW

### Step 1: Upload Base Model
**URL:** kaedim3d.com  
**Format:** STL or OBJ  
**Size:** Under 50MB

### Step 2: Describe Enhancement
**Example prompts:**

**For helmet:**
```
Original: Optimus Prime helmet
Enhancement: 
- Add Cylon chrome visor slit
- Smooth the dome (remove Prime ridges)
- Keep Prime battle mask shape
- Material: Chrome + Red/Blue panels
- Style: Heroic but menacing
```

**For chest:**
```
Original: Optimus Prime truck grille chest
Enhancement:
- Add layered Cylon armor plating
- Keep truck grille detail
- Add glowing Energon lines (recessed)
- Material: Chrome grille, Blue sides
- Style: Mechanical, functional
```

**For arms:**
```
Original: Robotic arm STL
Enhancement:
- Bulkier (Prime strength)
- Sleek joints (Cylon efficiency)
- Red upper arm, Blue forearm
- Chrome elbow pistons
- Material: Industrial metal
```

### Step 3: Select Variations
- Generate 3-5 variations
- Compare side-by-side
- Select best 2-3

### Step 4: Export
**Format:** STL (binary)  
**Scale:** Verify 1:6 ratio  
**Save:** To `CYLON_PRIME/parts/`
