# Done & Do's Convention
## AOCROS Project Management

**Format:** `Done_and_Dos_YYYY-MM-DD.md`

**Naming:** "Done" items first (completed work), "Do's" (todo items) second.

**Principle:** Acknowledge completed work before facing the remaining work.

- See wins first
- Builds momentum
- "Do's" are always waiting at the bottom
- Psychological: acknowledged before required
