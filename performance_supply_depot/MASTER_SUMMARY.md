## Quick Start (VPS)

```bash
# 1. Update system
sudo apt update && sudo apt upgrade -y

# 2. Install Docker
curl -fsSL https://get.docker.com | sh

# 3. Install Docker Compose
sudo apt install docker-compose

# 4. Clone config
git clone https://github.com/performancesupplydepot/ops.git

# 5. Start services
cd ops && docker-compose up -d

# 6. Check status
docker ps
```
