Key Design Principles
---------------------
1. Pi 5 = Brain (LLM + GPIO)
2. Android = Senses (camera/mic/speaker)
3. Local network only (no cloud)
4. Three-layer memory (con/subcon/uncon)
5. Owner signature on all writes
6. Fiduciary chain: User → Tappy → OpenClaw → Mylzeron

Governance
----------
User          (The Player)
  ↓
Tappy Lewis   (COO + Fiduciary)
  ↓
OpenClaw      (Coordination)
  ↓
Mylzeron      (Physical embodiment)
  ↓
Pi 5          (GPIO/Servos)
  ↓
Servos        (Physical motion)

Android Bridge Protocol
-----------------------
Camera ───HTTP POST──→ /sensor/camera
Mic ─────HTTP POST──→ /sensor/audio
Speaker ←─WS───────── /ws (audio commands)
Servos ←─WS───────── /command
