# Dusty Project

Cross-chain cryptocurrency wallet that consolidates dust across multiple networks.

## Quick Start

1. Open in DroidScript
2. Copy `src/dusty.js` into a new project
3. Run on Android device/emulator

- Staking with NFT rewards

- DApp browser
