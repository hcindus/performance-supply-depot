# Dusty - Cross-Chain Crypto Wallet

## Project Overview
Dusty is a cross-chain cryptocurrency wallet focused on consolidating crypto dust across multiple networks (Bitgert, Ethereum, Binance Smart Chain, Solana, etc.), providing low-cost transactions via Bitgert's high-speed, low-fee infrastructure.

## Core Features
- **Multi-Network Support**: Scan and manage assets across Bitgert, Ethereum, Binance, Solana, Arbitrum, Polygon
- **Dust Consolidation**: Aggregate small balances into main wallet
- **Cross-Chain Bridge**: Transfer assets between chains
- **Staking**: Stake assets for APY and NFT rewards
- **DApp Browser**: Access decentralized applications
- **Transaction History**: Track all operations
- **Settings**: Manage API keys for exchanges

## Technical Details
- **Platform**: DroidScript (Android mobile app)
- **Language**: JavaScript
- **UI Framework**: DroidScript UI components
